My skills.

As an aspiring computer science student in the modern world, I'm learning and mastering web technologies, programming fundamentals, and languages. Check out what I know.

Java

A main part of my education at CSULA was learning Java. Although some might say the language is outdated, it's a great foundation for programming in general, and through the Java series I've mastered OOP, the intricacies of data structures, and many, many types of sorting algorithms.

C#

As a gamer, I became interested in how to make games myself. So I invested time into learning how to use the Unity game engine, which itself uses the C# language. C# is quite similar to Java, but has a few key differences that set it apart from it, and in some cases make it better.

HTML, CSS, and JavaScript

When I initially became interested in programming, I used free online courses to learn the fundamentals of web programming. With that and a little gritty 'Inspect Element' curiosity, I have become fluent in the building blocks of the internet.

GitHub

Recently, I've started to collaborate with other students on a software project. This has essentially forced me to figure out and learn how to use GitHub and Git to properly implement version control for our project. Now, I couldn't imagine living without it!
